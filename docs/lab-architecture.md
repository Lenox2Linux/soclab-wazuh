## Lab Components

### Wazuh Server
- Repurposed Mac mini
- Rebuilt with Ubuntu Server
- Running Wazuh through Docker containers
- Used for log collection, monitoring, and dashboard access

### Testing Machine
- Repurposed 2012 MacBook Pro
- Running Ubuntu 22.04 LTS (Jammy Jellyfish) as the main operating system
- Hosting a Kali Linux virtual machine
- Used for scanning, packet analysis, and security testing exercises

### Monitored Endpoint
- M1 MacBook Air
- Running VMware
- Hosting an Ubuntu 24.04 LTS (Noble Numbat) virtual machine
- Used to generate system activity and logs visible in Wazuh

### Workstation
- Lenovo ThinkPad T14
- Used for administration, dashboard access, and project documentation
